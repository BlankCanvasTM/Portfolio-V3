Sonic New Adventures (Python & C#)

This project was my first official dive into game development. After I had spent some time watching some tutorials, I was ready to use Python and the pygame module to create my first game. I chose to make a Sonic clone because growing up Sonic played a very important role in my life and I felt this was a good idea to start with.

Splash Screen

Main Menu

Gameplay

As my first piece of game dev work I was quite proud of what I had created. I had coded everything from scratch and used assets that I could find online. However since I was making everything from the ground up I felt very limited with what python could achieve and I was itching to do more and challenge myself. That is when I heard of the Unity the game engine. I made the decison to remake the entire project using c#.

The Unity remake allowed me to achieve way more than I ever thought would be possible with Python. I used photoshop to make all the UI elements, I learned more about animations, work flow and scope. I was able to implement new features and really cemented my love for developing games.
